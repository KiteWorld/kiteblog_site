# 供应链管理系统「SCM」

  • 项目背景：供应链管理系统是一个基于 Vue 的大型项目，有十几个大的模块。本人主要开发了三个模块，产品研发管理、供应商考核 和 精准来货(基于原来的公司的项目开发，并非从零搭建)

  • # 产品研发管理

    • 模块背景：主要功能就是跟踪产品研发的进度以及任务多级分解，其实就是一个 PLM + WBS 部分功能混合的项目
    • 工作: 项目前端主开发
      • Webpack 配置 Vue 多入口
    • 收获：这是本人入职以来第一个开发的项目，受益匪浅，第一次看到大型企业项目的全貌。无论对项目全局观，还是技术细节，还是业务需求的分析，甚至是多人合作的开发模式等等都让受益匪浅。但也深深得感受到身为外包人员无奈。例如：不敢对公用组件进行改造、不敢给项目添加新的插件或者依赖等等。这主要是为划清责任边界问题，所以只能重复"造轮子"，蹑手蹑脚，很多时候施展不开。

  • # 精准来货

    • 模块背景：更精准地监控和协调公司与供应商之间的供需关系。
    • 工作: 项目前端由本人一人完成

  • # 供应商考核

    • 模块背景：主要是对供应商不同维度进行考核，并对供应商进行打分和归类。以此支撑来选择更好的供应商和是否需要赔付
    • 工作: 项目前端由本人一人完成
  • 时间：2020 - 07 ~ 2021 - 04
